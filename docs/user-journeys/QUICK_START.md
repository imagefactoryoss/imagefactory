# User Journeys - Quick Start Guide

This guide helps new users identify the role-specific documentation that matches how they use Image Factory.

→ Read [OWNER_JOURNEY.md](OWNER_JOURNEY.md)

- Billing and security

**Managing a specific project?**

- Project administration

## What’s in Each Document

Every journey document contains:

1. **Role Overview** - What this role does and their key responsibilities
2. **Permissions Matrix** - Exactly what they can and cannot do
3. **Getting Started** - First steps when joining
4. **Daily/Weekly Workflows** - Typical activities and rhythm
5. **Core User Journeys** - 3-5 detailed workflow examples with steps
6. **UI & Navigation** - Where to find things in the interface
7. **Key Features** - What's most important for this role
8. **Quick Actions** - Common tasks and keyboard shortcuts
9. **Scenarios & Examples** - Real-world usage patterns with time estimates
10. **Success Metrics** - How to know you're succeeding in this role

## First Time Here?

### Step 1: Identify Your Role
Use the "Are you..." section above ⬆️

### Step 2: Read Your Journey Document
- **New users**: Read in order (20-30 minutes)
- **Returning users**: Jump to the section you need
- **Managers**: Share relevant sections with your team

### Step 3: Reference as Needed
Bookmark the document and come back whenever:
- You need a reminder of what you can do
- You're onboarding someone new
- You have questions about permissions
- You want to learn about a specific workflow

### Step 4: Explore Related Documents
Each journey links to deeper feature guides and technical references for the workflows covered in that role.

## Reading Tips

- Use the table of contents in each guide to jump to the workflow you need.
- Search within the page with `Ctrl+F` or `Cmd+F` if you already know the task name.
- Treat the quick start as a routing guide, then use the role-specific document as your working reference.

---

## Still Need Help?

If you can't find what you're looking for:

1. **Search your role's document** - Use Ctrl+F or Cmd+F
2. **Check the FAQ section** - At the bottom of each journey
3. **Ask your team lead** - They can point you to the right section
4. **Contact support** - See your role's document for support links
